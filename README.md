## 💡 我们的初心 (The Vision)
在线下实体行业（如餐饮快餐、美甲店、理发店等），传统的 **“储值办卡”** 模式长期存在一个致命的痛点：**信任黑盒**。
顾客希望能享受到充值带来的高性价比优惠，但常常因为担心“老板卷款跑路”而被打退堂鼓。

**NeverRunaway** 应运而生。这是一个专为商业区、社区生活服务的实体小微企业设计的去中心化储值发卡 DApp平台。
我们利用以太坊智能合约的特性，将用户的储值金**锁定在链上（USDC）**：
- 顾客到店消费，通过核销码授权，资金才会变成商家可提取的利润。
- 如果商户关门倒闭，未核销的余额将支持一键原路退回！

我们希望让街坊邻居闭着眼就能放心享受长期的储值折扣。

## 🚀 当前 MVP 版本工作 (Current MVP)

目前本项目处于第一版的最小可行性产品（MVP）阶段，我们已经完成了全链路的闭环打通：

- **⛓️ Smart Contracts (Solidity)**:
  - 已部署于 Ethereum `Sepolia` 测试网。
  - 采用 `Factory` 工厂模式，允许商家一键零门槛发布自己独立的「储值资金池合约」。
  - 核心逻辑支持：用户抵押稳定币储值(Deposit)、商户核销扣款(Consume)、商户独立资金池提现(Withdraw) 以及超时违约的用户一键退款(Refund)。
  
- **🖥️ Backend (Flask + SQLite)**:
  - 极简且高效的轻量级 Python 后端。
  - 处理商户资质录入门店信息流转，以及消费核销码的安全生成与离线双重校验。
  
- **📱 Frontend (Vanilla HTML + JS + Tailwind)**:
  - 深色系的“玻璃拟物风（Glassmorphism）”UI 质感。
  - 完全实现了一键式中英双语切换。
  - 为【商家】与【消费者】分设两大独立看板，前端直连 MetaMask 插件唤醒所有的钱包过账授权。

## 🛠️ 如何在本地运行 (How to Run)

只需几步即可在本地跑起整个核心业务逻辑：

1. **环境准备**
   确保你安装了 Python (3.7+) 并且在浏览器中装有 MetaMask 插件，网络须切换至 `Sepolia` 测试网。

2. **启动后端服务**
   ```bash
   # 安装依赖
   pip install flask flask-cors
   
   # 启动服务 (默认将在 localhost:5000 运行并自动生成 SQLite 数据库)
   python app.py
   ```

3. **启动前端界面**
   在 `frontend` 文件夹下启动一个简单的 HTTP 服务器（切忌用 file:/// 直接打开）：
   ```bash
   cd frontend
   python -m http.server 3000
   ```
   随后打开浏览器访问 `http://localhost:3000` 即可开始测试完整的储值发卡交互！
